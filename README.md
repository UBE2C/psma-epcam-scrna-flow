# psma-epcam-scrna-flow <br />
This is a repository linked to the manuscript [**EpCAM–PSMA: Potential predictors of treatment outcomes for PSMA-targeted alpha therapies in metastatic castration-resistant prostate cancer](https://www.cell.com/molecular-therapy-family/oncology/fulltext/S2950-3299(26)00019-6)
Bakos, Gábor et al. Molecular Therapy Oncology, Volume 34, Issue 1, 201143** <br />
<br />

## Description
This repository contains the R scripts and analysis pipelines used for the analysis of data collected during Fluorescent-activated cell sorting (FACS) and single-cell-RNA-sequencing (scRNA-seq) experiments. More specifically, you will find the data-processing and plotting scripts for flow cytometry and single-cell RNA-seq analyses and additional data files generated during the analysis process. <br />
